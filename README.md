# OnStage
Repositório para o trabalho de APS de desenvolvimento de protótipo de sistema de compra e venda de ingressos online. 


Nome da página/marca: <b>OnStage</b><Br><Br>
1️⃣ Página Inicial – Tela Principal do Sistema<Br>

A primeira imagem apresenta a página inicial do sistema OnStage, onde o usuário pode visualizar os principais conteúdos disponíveis na plataforma. No topo da página há um menu de navegação com a logo do site, barra de pesquisa e botão de login/perfil.<Br>

A página contém seções organizadas como:<Br>

Eventos em Alta – mostra os filmes ou eventos mais populares.<Br>

Filmes em Alta – exibe os filmes mais assistidos.<Br>

Eventos Próximos – apresenta eventos que acontecerão em breve.<Br>

Estreando – mostra os lançamentos recentes.<Br>

Cada filme ou evento possui uma imagem e um botão "Comprar Ingresso", que direciona o usuário para a página de detalhes e sessões disponíveis.<Br>

Essa página funciona como o painel principal de navegação do usuário, facilitando o acesso rápido aos conteúdos do sistema.<Br><Br>

2️⃣ Menu de Perfil do Usuário<br>
A tela de login permite que o usuário acesse sua conta no sistema OnStage de forma simples e rápida.<br>

Nessa página, o usuário deve informar seus dados de acesso, como e-mail e senha cadastrados anteriormente.<br>

O layout é organizado de forma centralizada, facilitando a visualização e o preenchimento dos campos.<br>

Após inserir as informações corretamente, o usuário pode clicar no botão de entrar para acessar o sistema.<br>

Caso o usuário ainda não possua uma conta, ele pode utilizar a opção de cadastro disponível na página.<br>

Essa tela é importante para garantir a segurança e o controle de acesso às funcionalidades do site.<br>

3️⃣Página – Tela de Cadastro

A tela de cadastro permite que novos usuários criem uma conta no sistema OnStage.<br>

Nessa página, o usuário deve preencher seus dados pessoais, como nome, e-mail e senha.<br>

As informações cadastradas são utilizadas para identificar o usuário dentro do sistema.<br>

O layout é organizado de forma simples e intuitiva para facilitar o preenchimento dos campos.<br>

Após preencher os dados corretamente, o usuário pode finalizar o cadastro para acessar o site.<br>

Essa página é importante para permitir que o usuário utilize funcionalidades como compra de ingressos e acesso ao perfil.<br>

4️⃣Página de Sessões do Filme

A terceira imagem apresenta a página de sessões do filme, onde o usuário pode visualizar informações detalhadas sobre o filme selecionado.<br>

Nessa página são exibidos:<br>

Imagem do filme.<br>
Título.<br>
Classificação indicativa.<br>
Duração.<br>
Gênero.<br>
Sinopse.<br>

Além disso, o usuário pode visualizar os dias disponíveis e horários das sessões, podendo escolher o horário desejado.<br>
Essa página funciona como a etapa intermediária antes da escolha dos assentos, permitindo ao usuário selecionar a sessão desejada.<br>

5️⃣ Página de Seleção de Assentos

A quarta imagem mostra a tela de seleção de assentos, onde o usuário escolhe os lugares disponíveis para a sessão selecionada.<br>

Os assentos são representados visualmente, permitindo que o usuário:<br>

Veja os assentos disponíveis.<br>
Veja os assentos ocupados.<br>
Selecione os assentos desejados.<br>

Após a seleção, o usuário pode avançar para a próxima etapa da compra.<br>

Essa página é responsável pela escolha dos lugares dentro da sala de cinema.<br>

 6️⃣Página – Seleção de Ingressos<BR>

A página de seleção de ingressos permite que o usuário escolha a quantidade de ingressos desejados para a sessão selecionada.<br>

Nessa etapa, o usuário pode definir quantos ingressos serão do tipo inteira ou meia-entrada.<br>

O sistema organiza as opções de forma clara para facilitar a escolha antes do pagamento.<br>

Após selecionar a quantidade e os tipos de ingressos, o usuário pode avançar para a próxima etapa.<br>

Essa página é importante para calcular o valor total da compra de forma automática.<br>

7️⃣ Página – Forma de Pagamento<BR>
A página de pagamento permite que o usuário escolha a forma de pagamento para finalizar a compra.<br>

Nessa tela são exibidas opções como cartão de crédito, débito, Pix e outras formas disponíveis.<br>

O layout apresenta as opções de forma visual e organizada para facilitar a seleção.<br>

Após escolher a forma de pagamento, o usuário pode concluir a compra dos ingressos.<br>

Essa página representa a etapa final do processo de compra no sistema OnStage.<br>
